# AI Forum Bangladesh Website

A modern, responsive website designed and developed by **Softrare Labs** for **AI Forum Bangladesh**, an independent non-profit organization helping prepare the next generation for the Artificial Intelligence era.

The website presents the organization’s mission, initiatives, advisors, team, and hackathon programs through a clean digital experience built for trust, clarity, and future growth.

---

## Project Overview

**AI Forum Bangladesh** needed a professional web presence that could communicate its purpose, showcase its work, and support a growing community focused on responsible and accessible AI education.

Softrare Labs created a polished, scalable website experience that reflects the organization’s vision: empowering youth with the mindset, mentorship, and opportunities needed to thrive in an AI-driven world.

> **Note:** This repository is used as a public project showcase. The source code is not included due to privacy, confidentiality, and intellectual property protection.

---

## Goals & Challenge

The project goal was to build a website that could clearly communicate the mission of AI Forum Bangladesh while maintaining a professional, modern, and trustworthy appearance.

Key objectives included:

- Present the organization’s mission in a clear and inspiring way.
- Highlight initiatives, advisors, team members, and hackathon information.
- Build a responsive experience for desktop, tablet, and mobile users.
- Create a clean interface suitable for a non-profit, education, and technology-focused audience.
- Support performance, accessibility, and long-term scalability.
- Keep the structure flexible for future content and program updates.

---

## Softrare Labs Solution

Softrare Labs designed and developed a refined web experience focused on clarity, usability, and credibility.

The interface uses a modern visual system, structured content sections, smooth interactions, and responsive layouts to make the organization’s story easy to understand. The website was built with a scalable front-end architecture, allowing the platform to grow as AI Forum Bangladesh expands its programs and community impact.

---

## Key Features

- Fully responsive website design
- Modern homepage with clear mission-focused messaging
- Dedicated sections for initiatives, advisors, team, and hackathon details
- Clean UI components with professional modal cards
- Smooth animations and transitions
- Admin-friendly content structure
- Optimized image handling for faster page loading
- Performance-conscious front-end implementation
- Accessible layout and readable content structure
- Scalable architecture for future updates

---

## Tech Stack

- **Next.js 14** - React framework for fast, scalable web development
- **Tailwind CSS** - Utility-first styling for clean and consistent UI
- **Framer Motion** - Smooth, subtle animations and transitions
- **EmailJS** - Backend-less communication and form handling
- **Next.js Image Optimization** - Optimized images for improved performance
